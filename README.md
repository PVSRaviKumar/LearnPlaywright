# LearnPlaywright

A small repository for practicing JavaScript and Playwright concepts.

## Repository Structure

- `chapter_01_Basics/` - Basic Playwright examples and setup verification snippets.
- `chapter_02_JavaScript_Concepts/` - JavaScript basics and related code samples.
- `chapter_03_Identifiers_Literals/` - Identifier rules and literals in JavaScript.
- `chapter_04_JavaScript_concepts/` - JavaScript concepts such as scoping, hoisting, and function behavior.
- `chapter_05_Literal/` - Literal types, number systems, templates, and string examples.
- `chapter_06_Operators/` - JavaScript operators, comparisons, logical expressions, and null/undefined behavior.

## Usage

This repository contains example JavaScript files for learning and experimenting with JavaScript fundamentals and Playwright basics. Open a chapter folder to review the sample scripts.

## Notes

- The repository is organized by chapter/topic.
- Each file is a standalone example or tutorial snippet.
