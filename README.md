# LearnPlaywright

LearnPlaywright is a hands-on practice repository for JavaScript, TypeScript, and Playwright. It contains small, topic-based example files that are useful for daily revision, interview preparation, and step-by-step learning.

## What this repository covers

- JavaScript fundamentals and core concepts
- Operators, control flow, loops, arrays, functions, strings, and objects
- Object-oriented programming topics such as classes, encapsulation, inheritance, and polymorphism
- TypeScript basics and interface-based examples
- Playwright basics for browser automation practice

## How to use this repo

1. Open the project in your editor.
2. Navigate to the chapter folder that matches the topic you want to practice.
3. Run JavaScript files with Node.js, for example:
   ```powershell
   node Chapter_07_if_else/48_if_else.js
   ```
4. Compile TypeScript files with the TypeScript compiler, for example:
   ```powershell
   npx tsc Chapter_27_TypeScript_Interface/201_TypeScript_Interface.ts
   ```
5. Modify the examples and experiment with different inputs to reinforce the concepts.
